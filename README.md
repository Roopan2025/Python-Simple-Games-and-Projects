# Python Console Projects Collection

A collection of beginner-friendly Python console applications developed to strengthen programming fundamentals, problem-solving skills, and object-oriented programming concepts. These projects cover user input handling, loops, conditional statements, functions, randomization, file handling, encryption techniques, and basic calculations.

13. Shopping Cart System

15. Temperature Converter
16. Weight Converter
